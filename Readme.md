An AI-powered speech-to-speech translation application that converts spoken English into Telugu, Hindi, and French audio.  
The system uses modern AI services for transcription, translation, and text-to-speech, all wrapped in an easy-to-use Gradio web interface.


Features
- Voice input using microphone
- Speech-to-text transcription with AssemblyAI
- Automatic translation to:
  - Telugu
  - Hindi
  - French
- Natural-sounding multilingual speech using ElevenLabs
- Simple interactive UI built with Gradio

- Python
- Gradio – Web interface
- AssemblyAI – Speech-to-text
- ElevenLabs – Text-to-speech
- UUID & Pathlib – File handling
